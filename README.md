# 📝 DeskNoteX - 桌面便签任务管理

一个简洁美观的桌面便签应用，基于 Python + PyQt5 构建，支持任务管理、分类标签、优先级设置和提醒通知。

## ✨ 核心功能

| 功能 | 说明 |
|------|------|
| **任务管理** | 创建、编辑、删除任务，支持搜索筛选 |
| **分类标签** | 自定义分类，按颜色区分不同类型 |
| **优先级** | 高/中/低三级优先级，醒目提示 |
| **截止日期** | 设置任务截止时间，到期提醒 |
| **状态追踪** | 全部/进行中/已完成，清晰分类 |
| **数据统计** | 任务完成率、分类分布可视化 |
| **主题切换** | 浅色/深色模式，支持自定义主题色 |
| **边缘贴边** | 窗口自动贴边收纳，不占桌面空间 |
| **系统托盘** | 最小化到托盘，后台常驻提醒 |
